Phase 1 — Python Proficiency + Core AI Stack

Month 1 · Weeks 1–4 · ~56 hours total
▾
Week 1 — Python Power-Up (2–3h/day)
Python Intermediate Bootcamp — OOP, Decorators, Generators
Focus: Classes, dunder methods, decorators, generators, iterators, context managers. You already know basics — fast-track to advanced Python used in AI frameworks.
Critical Python
3h · Days 1–2
Python AsyncIO — async/await, event loops, concurrent tasks
Core for agentic AI systems. Build: an async API caller with rate limiting + timeout handling. This is required for LangGraph agentic loops.
Critical Python
4h · Days 3–4
Python for Data — NumPy, Pandas, Matplotlib basics
Required for ML/AI work. Focus on DataFrame ops, data cleaning, vectorized ops. Don't go deep — just get functional.
High Python
3h · Days 5–6
FastAPI from scratch — Routes, Pydantic models, async endpoints, middleware
Build: A FastAPI service that exposes an AI endpoint (wraps OpenAI API). Deploy locally with uvicorn. You'll use this pattern for every AI microservice.
Critical FastAPI
4h · Days 6–7
Week 2 — Vector Databases + Embeddings
Embeddings deep-dive — What they are, how to generate, similarity search
Use OpenAI text-embedding-3-small or HuggingFace sentence-transformers. Build a semantic search over 100 documents. Understand cosine similarity vs dot product.
Critical Embeddings
3h · Days 8–9
Pinecone — Setup, upsert, query, metadata filtering
Free tier available. Ingest 500+ documents, query by semantic similarity, apply metadata filters. This replaces "embedded vector databases" on your resume with a real name.
Critical Vector DB
3h · Days 9–10
pgvector — Run vector search inside PostgreSQL
Install pgvector extension, store embeddings in Postgres, run similarity queries. Bridges your existing SQL skills to AI. Highly valued in enterprise settings.
High Vector DB
2h · Day 11
Mini-Project: Resume ATS Semantic Search Engine
Use your own resume HTML + JDs → embed → store in Pinecone → query by semantic similarity. You already have the ATS app — enhance it with real vector search instead of TF-IDF.
Project Resume Impact
5h · Days 12–14
Week 3 — LangGraph + Agentic AI
LangGraph fundamentals — Nodes, edges, state, conditional routing
LangGraph is a state machine for AI agents. Learn: StateGraph, add_node, add_edge, compile, invoke. Build a simple 3-node agent (Plan → Execute → Reflect).
Critical LangGraph
4h · Days 15–16
ReAct + Plan-and-Execute reasoning patterns
Implement ReAct (Reason + Act) loop. Build an agent that uses tools (web search, calculator, DB query) to answer multi-step questions.
Critical Agentic AI
4h · Days 17–18
MCP + LangGraph integration
You already use MCP at BNY Mellon! Connect LangGraph agents to MCP-defined data sources (Snowflake, Oracle). This bridges your real work to the open-source framework.
Critical You have MCP exp!
3h · Days 19–20
Mini-Project: Job Application AI Agent
Build a LangGraph agent that: reads a JD → analyzes skills → queries Pinecone for resume chunks → generates a tailored cover letter. Extend your ATS app.
Project Portfolio
6h · Days 20–21
Week 4 — Advanced RAG Architectures
Chunking Strategies — Fixed, semantic, hierarchical, late chunking
Most engineers use naive fixed chunking. Learn: recursive character splitting, semantic chunking with embeddings, sentence-window retrieval. Huge impact on RAG quality.
Critical RAG
3h · Days 22–23
Hybrid Search — BM25 + semantic + RRF (Reciprocal Rank Fusion)
Pure semantic search misses keyword matches. Learn hybrid: BM25 for keyword, embeddings for semantic, fuse with RRF. Implement with LangChain EnsembleRetriever.
Critical RAG
3h · Days 24–25
Re-ranking — Cohere Rerank, cross-encoders, ColBERT
Retrieve 20 docs → re-rank to top 5 → pass to LLM. This dramatically improves answer quality. Cohere Rerank has a free tier — start there.
High RAG
2h · Day 26
Capstone: Enterprise RAG System with FastAPI + Pinecone + LangGraph
Full pipeline: ingest docs → chunk → embed → Pinecone → hybrid search → re-rank → LangGraph agent answers. Expose via FastAPI. Deploy with Docker. Add to GitHub + resume.
Capstone Portfolio ⭐

🐍 Python Proficiency Track
Your starting point: You know Python basics from AWS Bedrock/SageMaker work. This track fast-tracks you to the advanced Python used in AI frameworks like LangChain, LangGraph, HuggingFace, and FastAPI.

Level 1 — Intermediate (Week 1)

  • OOP Advanced — dataclasses, abstract classes, mixins
  • Decorators — @property, @classmethod, custom decorators
  • Generators — yield, generator expressions, lazy evaluation
  • Context Managers — with statement, __enter__/__exit__
  • Type Hints — typing module, Pydantic models
  • Resource: Corey Schafer Python Tutorials (YouTube)

Level 2 — Async Python (Week 1)

  • async/await — coroutines, event loop
  • asyncio.gather() — parallel async tasks
  • aiohttp — async HTTP client for API calls
  • Timeout handling — asyncio.wait_for()
  • Concurrency vs Parallelism — when to use threading vs asyncio vs multiprocessing
  • Resource: ArjanCodes AsyncIO series (YouTube)

Level 3 — Data Libraries (Week 1–2)

  • NumPy — arrays, broadcasting, vectorized ops
  • Pandas — DataFrames, groupby, merge, apply
  • Matplotlib/Seaborn — basic plotting for ML results
  • scikit-learn — pipelines, transformers, cross-validation
  • Resource: Keith Galli Pandas (YouTube), CS50 Python

Level 4 — Python for AI Frameworks (Week 2–3)

  • Pydantic v2 — data validation, model config, validators
  • FastAPI patterns — dependency injection, background tasks, WebSockets
  • dotenv + config management — settings classes with Pydantic
  • Logging — structured logging for AI apps
  • Resource: FastAPI official docs + TechWithTim FastAPI

Level 5 — Python Testing (Ongoing)

  • pytest — fixtures, parametrize, mocking
  • pytest-asyncio — testing async code
  • unittest.mock — mock LLM responses in tests
  • Coverage — pytest-cov reports (you know this from Java/JUnit!)
  • Resource: ArjanCodes Testing Python (YouTube)

Python for LLM/AI (Phase 2–3)

  • tiktoken — token counting, context window management
  • LangChain LCEL — Python-native chain composition syntax
  • Streaming responses — yield from LLM stream, SSE
  • Memory management — avoid OOM with large embedding batches
  • Resource: LangChain docs + Greg Kamradt notebooks

📊 Python Practice Projects (Build these)

Week 1 Project

  • CLI tool that reads a JD text file and scores it against your resume using Python (replicate your ATS backend logic in pure Python)
  • Uses: OOP, file I/O, dataclasses, type hints

Week 3 Project

  • Uses: FastAPI, pdfplumber, httpx, Pinecone SDK, Pydantic
